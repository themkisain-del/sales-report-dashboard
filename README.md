
# sales-report-dashboard
This project contains a Power BI Sales Dashboard designed to analyze sales performance across multiple dimensions.   It provides insights into revenue, profit, order volume, product categories, payment methods, and sales channels.   
It provides insights into:
- Revenue & Profit
- Order Volume
- Product Categories
- Payment Methods
- Sales Channels
- Monthly Trends

## 🖼️ Dashboard Highlights
- **Net Sales:** ₹29.81M  
- **Profit:** ₹8.01M  
- **Orders:** 500  

### Key Visuals
- Line chart: Monthly Sales Trend  
- Bar chart: Sales by Category (Electronics highest)  
- Donut chart: Payment Method distribution (Debit Card, UPI, Net Banking, Cash, Credit Card)  
- Donut chart: Sales Channel distribution (Retail, Distributor, Online, Direct Sales)  

## 🎯 Purpose
The dashboard helps businesses:
- Track revenue and profitability
- Identify top‑performing categories
- Understand customer payment preferences
- Evaluate sales channel effectiveness

## ⚙️ Tools Used
- **Power BI Desktop**
- Dataset: Sales transactions (Orders, Categories, Payment, Channels)

## 🚀 How to Use
1. Download the `.pbix` file from this repository.  
2. Open in **Power BI Desktop**.  
3. Apply filters (Quarter, Category, etc.) to explore insights.
